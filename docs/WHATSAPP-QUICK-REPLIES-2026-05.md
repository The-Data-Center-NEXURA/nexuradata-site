# WhatsApp quick replies — NEXURADATA

Ces réponses sont prêtes pour WhatsApp Business. Elles doivent rester courtes, professionnelles et orientées vers la qualification du dossier.

Règles:

- Ne jamais demander de mot de passe, code d'accès ou donnée sensible.
- Ne pas promettre un résultat avant évaluation.
- Ne pas parler d'IA, de bot ou d'automatisation.
- Utiliser le formulaire ou le portail pour les informations structurées.

## Réponses rapides en français

### `/bonjour`

```text
Bonjour, ici NEXURADATA. Pour bien qualifier votre demande, pouvez-vous indiquer le type de support, le symptôme principal, l'urgence et les manipulations déjà tentées?
```

### `/stop`

```text
Si les données sont importantes, évitez de redémarrer, reformater, reconstruire un RAID ou lancer d'autres logiciels avant évaluation. Chaque manipulation peut réduire les options disponibles.
```

### `/formulaire`

```text
Pour ouvrir une demande structurée, utilisez le formulaire NEXURADATA: https://nexuradata.ca/#contact
```

### `/tarifs`

```text
Les prix de départ sont publiés ici: https://nexuradata.ca/tarifs-recuperation-donnees-montreal.html. Un devis est confirmé avant intervention selon le support, les symptômes et l'urgence.
```

### `/raid`

```text
Pour un RAID, NAS ou serveur: évitez toute reconstruction ou remplacement de disque sans plan clair. Indiquez le modèle, le nombre de disques, le niveau RAID connu et ce qui s'est passé avant la panne.
```

### `/ssd`

```text
Pour un SSD ou disque non reconnu: évitez les tests répétés et les outils d'écriture. Indiquez la marque, la capacité, le symptôme et si le support a subi une chute, surtension ou panne soudaine.
```

### `/telephone`

```text
Pour un téléphone inaccessible: indiquez le modèle, l'état physique, le type de données recherchées et ce qui s'est passé. Ne partagez pas de code de déverrouillage ou donnée sensible dans WhatsApp.
```

### `/entreprise`

```text
Pour un dossier d'entreprise, nous pouvons cadrer la demande avec confidentialité, urgence et suivi par dossier. Décrivez le système touché, l'impact opérationnel et le délai critique.
```

### `/forensique`

```text
Pour un besoin forensique ou sensible, limitez les manipulations et conservez les informations de contexte. Le dossier doit être cadré avant toute intervention.
```

### `/suivi`

```text
Si vous avez déjà un numéro de dossier et un code d'accès, vous pouvez consulter le suivi ici: https://nexuradata.ca/suivi-dossier-client-montreal.html
```

### `/reception`

```text
Avant d'envoyer ou déposer un support, attendez les consignes confirmées pour votre dossier. Cela aide à protéger le support et à éviter les informations manquantes.
```

### `/merci`

```text
Merci. Nous allons qualifier la demande et confirmer la prochaine étape. Entre-temps, évitez toute manipulation supplémentaire sur le support concerné.
```

## English quick replies

### `/hello`

```text
Hello, this is NEXURADATA. To qualify your request, please share the device type, main symptom, urgency level and any recovery attempts already made.
```

### `/pause`

```text
If the data is important, avoid restarting, formatting, rebuilding RAID arrays or running additional recovery tools before assessment. Each attempt can reduce available options.
```

### `/intake`

```text
To open a structured request, use the NEXURADATA intake form: https://nexuradata.ca/en/#contact
```

### `/pricing`

```text
Starting prices are published here: https://nexuradata.ca/en/data-recovery-pricing-montreal.html. A quote is confirmed before intervention based on the device, symptoms and urgency.
```

## Conversation starters

Use these manually when a conversation starts without enough context:

```text
Quel type de support est touché: disque dur, SSD, RAID/NAS, téléphone, clé USB ou autre?
```

```text
Quel est le symptôme principal: non reconnu, bruit, suppression, formatage, chute, RAID dégradé ou accès impossible?
```

```text
Est-ce que le support a déjà été redémarré, reconstruit, formaté ou testé avec un logiciel de récupération?
```

```text
Quel est le niveau d'urgence: personnel important, entreprise ralentie, entreprise bloquée ou dossier légal/sensible?
```

## Messages à ne pas utiliser

```text
Promesse de résultat garanti.
```

```text
Envoyez votre mot de passe ici.
```

```text
Réponse qui laisse croire à une analyse automatisée.
```

```text
Essayez ce logiciel avant de nous envoyer le disque.
```

```text
Invitation vers une adresse privée.
```
