# Launch Checklist

Points a finaliser avant publication publique complete:

- Confirmer la denomination sociale exacte de l'exploitant.
- Ajouter le NEQ si vous souhaitez le publier sur le site.
- Publier les coordonnees nominatives de la personne responsable de la protection des renseignements personnels.
- Confirmer la ligne telephonique publique.
- Configurer la boite `contact@nexuradata.ca`.
- Verifier les informations d'hebergement et les fournisseurs essentiels a publier.
- Valider les modalites de paiement effectivement acceptees.
- Verifier que tous les engagements affiches correspondent aux pratiques reelles.
